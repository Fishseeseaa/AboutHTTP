### Servlet线程安全问题
    void sal()
    {
      sal=...//修改局部变量
    }
##### 1.局部变量存入栈内。
##### 2.每个线程有不同的栈帧。
##### 3.不同的线程数据是分开的，无线程安全问题
    sal=0
    void service()
    {
      sal=...//sal为全局变量
    }
##### 1.成员变量存入堆中。
##### 2.堆中该数据只有一份。
##### 3.不同线程同时修改这一份数据有线程安全问题

### 解决方案：
1.不用成员变量
2.加锁。
